このリポジトリのドキュメントの整合性をチェックする。報告のみで、ファイルは変更しない。

1. README / AGENTS.mdと実態: 記載のコマンドが存在し動くか、ディレクトリ構成が実際のツリーと一致するか、依存の名前・バージョンがマニフェストと一致するか。
2. ADR(`docs/adr/`がある場合): `docs/adr/README.md`のインデックスとファイルの一致、番号の連番(欠番を報告)、非推奨ADRの明示。
3. 相互参照: ドキュメント間のリンクが解決できるか、削除・リネームされたファイルへの言及が残っていないか。

指摘を重要度別の箇条書きで日本語で報告する。問題がなければ「整合性チェック OK」と報告する。
