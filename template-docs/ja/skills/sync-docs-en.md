# sync-docs-enスキル(日本語訳)

日本語Markdownの英語版(`_en.md`)を同期させるスキル。`_en.md`対応ファイルを持つ日本語mdを編集した後、check-docs-en-sync hookにリマインドされたとき、英語版が必要な新規日本語ドキュメントを作成したときに使う。

ルール(AGENTS.md): 日本語の`<名前>.md`が正。`<名前>_en.md`を同じコミットで更新する。

1. 変更点を確認する: `git diff -- <名前>.md`(新規ならファイル全体)。
2. `<名前>_en.md`に同じ変更を反映する: 文章は自然に翻訳し、見出し・セクション順・コードブロック・コマンド・パス・固有名詞は同一に保つ(2ファイルを並べて比較できるように)。
3. `<名前>_en.md`がなければ作成する — ただし薄いポインタファイル(`CLAUDE.md`・`.github/copilot-instructions.md`)と実プロジェクトの個別ADRは日本語のみでよい。
4. 両ファイルを同じコミットに含める。
