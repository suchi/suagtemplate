# /merge-pr(日本語訳)

最終チェックのうえ、第1引数で指定されたPRをマージする。

## 前提条件(すべて確認する。省略禁止)

1. レビュー完了: 未解決スレッドがなく、最新のCopilotレビューが新規コメントなし
2. CIの全ジョブがグリーン
3. ユーザーがマージを明示的に承認している(ユーザーがこのコマンドを実行したことは承認とみなす)

前提条件を満たさない場合は、マージせずに停止して報告する。

## 手順

1. `gh pr merge <PR番号> --merge --delete-branch`
2. `git checkout main && git pull`
3. 第2引数(バージョンタグ)がある場合:
   a. `git tag -a <バージョン> -m "<バージョン>"`
   b. `git push origin --tags`
4. ローカルのフィーチャーブランチが残っていれば削除し、日本語で完了報告する。
