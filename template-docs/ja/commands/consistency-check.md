# /consistency-check(日本語訳)

このリポジトリのドキュメントの整合性をチェックする。報告のみで、ファイルは変更しない。

1. README / AGENTS.mdと実態: 記載のコマンドが存在し動くか、ディレクトリ構成が実際のツリーと一致するか、依存の名前・バージョンがマニフェストと一致するか。
2. 日英ペア: `<名前>_en.md`を持つすべての`<名前>.md`について、見出しが一致し英語版が古くなっていないか。
3. ADR(`docs/adr/`がある場合): `docs/adr/README.md`のインデックスとファイルの一致、番号の連番(欠番を報告)、非推奨ADRの明示。
4. 相互参照: ドキュメント間のリンクが解決できるか、削除・リネームされたファイルへの言及が残っていないか。

指摘を重要度別の箇条書きで日本語で報告する。問題がなければ「整合性チェック OK」と報告する。
