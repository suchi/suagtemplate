# /consistency-check(日本語訳)

このリポジトリのドキュメントの整合性をチェックして報告する。報告のみで、ファイルは変更しない。

## チェック項目

1. README / AGENTS.mdと実態:
   - 記載されたコマンド・スクリプト名が存在し動くか
   - ディレクトリ構成の記述が実際のツリーと一致するか
   - 依存関係の名前・バージョンがマニフェストと一致するか
2. 日英ペア:
   - `<名前>_en.md`を持つすべての`<名前>.md`について、見出し構造が一致し、英語版の内容が古くなっていないか
3. ADR(`docs/adr/`がある場合):
   - `docs/adr/README.md`のインデックスにすべてのADRファイルが載っており、載っているファイルがすべて存在するか
   - 番号が連番か(欠番を報告)
   - 非推奨ADRがインデックスで明示されているか
4. 相互参照:
   - ドキュメント間のリンクが解決できるか
   - 削除・リネームされたファイルへの言及が残っていないか

## 出力

指摘を重要度別の箇条書きで日本語で報告する。問題がなければ「整合性チェック OK」と報告する。
